## What this won't do

Directory submissions ALONE won't move the GSC needle. They build DR slowly (typically 3-6 months for new directories to pass meaningful link equity). The real value is:

1. **Branded search lift** — investors searching "AIWholesail" find consistent listings everywhere
2. **Comparison page rankings** — when AlternativeTo and G2 list us with PropStream/DealMachine/BatchLeads, those comparison pages start ranking for "PropStream alternative" type queries
3. **Compounding social proof** — each new listing reinforces legitimacy

Pair this with active content (blog automation already running) and outbound (cold email, podcast tour) for compound effect.
